# Roadmap

## Bootstrap

- Establish agent workflow, repository standards, and Claude orchestration.

## Foundation Slice

- Add npm workspaces for `apps/showcase`, `packages/ui`, and `packages/themes`.
- Build token output, light/dark support, one original violet theme, and initial components.

## Component Library

- Grow accessible, token-driven Vue Lynx components with stable JavaScript consumer APIs.

## Documentation and Showcase

- Provide JavaScript-first docs, examples, and a Vue Lynx showcase application.

## Admin Starter

- Create an original admin starter focused on practical usability and maintainable composition.
