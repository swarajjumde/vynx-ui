# Contributing

This project is in early bootstrap. Keep changes small, scoped, and tied to a task.

## Expectations

- Follow `ARCHITECTURE.md` and the standards in `docs/standards/`.
- Prefer JavaScript-first examples for consumers.
- Use TypeScript internally only when it improves maintainability without harming JavaScript use.
- Add focused tests for public behaviour.
- Do not copy code, assets, branding, or visual identity from other UI libraries or templates.

## Safety

- Do not publish packages, push Git branches, create remotes, or run destructive commands without explicit approval.
- Do not access secrets, `.env` files, credential stores, browser data, or authentication storage.
